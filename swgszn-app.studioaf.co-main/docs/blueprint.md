# **App Name**: Stoic AF

## Core Features:

- Daily Journaling: Secure, encrypted journaling with offline persistence and automatic autosaving of entries. Includes mood tracking and tagging.
- Adaptive Stoic Prompts: Receive daily personalized Stoic prompts based on virtue focus, mood, tags, and previous entries. Uses local engine with deterministic fallback for offline use.
- AI-Powered Weekly Reflections: Generate weekly summaries and insights from journal entries using Genkit and Gemini 1.5 Flash. This is a pro-only feature with input capped to control AI costs.
- Quick Capture: Enable quick journal entries with a global keyboard shortcut.
- Motivational Dashboard: Visualize progress with streaks, mood trends, and virtue focus radial charts. Track progress and stay motivated.
- AI Coaching: Button to ask a generative AI "tool" questions in real-time and receive support for implementing stoicism into daily habits. Adjust tone and focus within settings.
- Secure Data Storage: End-to-end encryption using Firestore for privacy and row-level security.

## Style Guidelines:

- Primary color: Desaturated blue (#4B90C8) to evoke calmness and reflection.
- Background color: Very light gray (#F8FAFC) for a clean, minimalist aesthetic in light mode, and dark gray (#1E293B) in dark mode.
- Stoic blue (#4B90C8) for primary actions
- Stoic dark (#1E293B) for dark surfaces, streak cards
- Stoic light (#F8FAFC) for app background
- Stoic gray (#64748B) for secondary text
- Body: 'Inter', sans-serif.
- Headings: 'Playfair display', serif.
- Use minimalist, line-based icons to represent virtues and actions, aligning with Stoic principles.
- Maximize readability with a max content width of 768px, rounded cards, and subtle shadows for depth.
- Incorporate smooth, subtle transitions with a duration of 150ms to enhance user experience without distraction.
